This Chinese Bookmarklets for Reading project provides browser-based JavaScript tools to assist with reading, including: 

∙ a pinyin webfont

∙ hundreds of pronunciation corrections to the webfont

∙ easy-to-access dictionary menu

∙ a tool for removing pinyin from familiar characters

∙ a brief text analyzer


These tools can be installed by creating a bookmark and pasting the code directly into the URL field.


It also includes two non-bookmarklet tools:

∙ page of tools to assist with paper texts

∙ a more detailed analyzer for copy-pasted texts
